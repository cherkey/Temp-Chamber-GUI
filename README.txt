README:

All three files are required to run the Temperature Chamber GUI:

thermotron2800_gui.bat
thermotron2800_gui.ps1
thermotron2800_chart.ps1

The .bat file is the gui launcher, the gui.ps1 is the main code, and the chart.ps1 is the code for the graphical chart display that the gui.ps1 references.

All three must be placed into the same directory in order to run correctly.
